How it works

Up and running in minutes

Four steps from install to a fully personalized AI clone that gets smarter over time.

1

Install in one command

The script detects your platform, checks dependencies, and sets up the engine at ~/.clone-engine.

2

Let the agent scan your files

The agent locally reads your dotfiles, configs, READMEs, and projects to understand how you think and work. Nothing is uploaded.

3

Get your CLI command

A shell alias is added for your name (e.g. alex). Run it from any directory to start a session.

4

Memory compounds over time

Every session is stored and indexed locally. The longer you use it, the more context it carries — all on your machine.
