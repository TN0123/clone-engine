How it works

Install in one command

The install script detects your platform, checks dependencies, and sets up the engine at ~/.clone-engine.

Answer the setup interview

10–15 minutes of honest questions about how you think, talk, disagree, and build. The answers become your clone's personality profile.

Get a personalized CLI command

A shell alias is added for your name (e.g. alex). Run it from any directory to start a session with your clone.

Memory compounds over time

Every session is stored and indexed. The longer you use it, the more context it carries — decisions, code, conversations.
